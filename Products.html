Example for HW #3

This web page provides examples that can be modified to help with the last questions in HW #3. Note the following:

  • The inputs in the first row have ids of "check1" and "num1"; for the second row, these are "check2" and "num2". This allows us to link the checkbox with the numeric input.
  • Clicking a checkbox automatically sets the corresponding numeric input to 23. See the processCheck function.
  • Clicking the button will 'read' the 3rd column and will display an alert showing the names of the items.
